# Скрипты backend

## seed_admin_data.py

Заполняет БД тестовыми данными для проверки админ-панели и пагинации:

- **24 пользователя** — `user1@test.local` … `user24@test.local`, пароль: `TestPass123!`
- **44 транзакции** — разные статусы и суммы
- **15 новостей**
- **12 email-рассылок**
- **44 записи журнала аудита**

**Требования:** БД должна быть инициализирована (таблицы создаются при первом запуске приложения или скрипта). Если пользователей ещё нет, скрипт создаёт 24 тестовых; при повторном запуске пользователи с такими email не дублируются.

**Запуск** (из каталога `backend`):

```bash
cd backend
python scripts/seed_admin_data.py
```

После выполнения во вкладках «Транзакции», «Новости», «Рассылки», «Журнал» будет достаточно данных для проверки пагинации (по 20 записей на страницу).
