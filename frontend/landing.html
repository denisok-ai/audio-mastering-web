Функции, которых нет
у конкурентов

LANDR и eMastered предлагают либо эталонный мастеринг, либо стриминг-превью — никто не объединяет всё сразу. Мы объединяем.

Эталонный мастеринг
Reference Track
Загрузите трек-эталон — система построит и применит корректирующую EQ-кривую через FFT-спектральный анализ. Ваш трек звучит как ваша любимая запись.
Адаптивная коррекция
Dynamic EQ — 8 полос
EQ срабатывает только когда нужно — реагирует на уровень сигнала в каждой полосе. В 3–5 раз прозрачнее многополосного компрессора. Устраняет резонансы и mud без потери характера.
Контроль динамики
Transient Designer
Независимый контроль атаки и сустейна через двойной envelope follower. Добавьте панч барабанам или смягчите атаку акустики. Нет ни в одном веб-сервисе мастеринга.
Очистка записи
Spectral Denoiser
Wiener-фильтр в частотной области удаляет фоновый шум, гул и артефакты записи. Оценка шумового профиля по медиане спектра — стабильна к транзиентам, не режет высокие.
NY-компрессия
Parallel Compression
Blend сильно сжатого сигнала с оригиналом. Сохраняет транзиенты и добавляет плотность — стандартная студийная техника для сочного, музыкального звука без "зажатости".
До публикации
Streaming Loudness Preview
Показывает как трек будет нормализован на Spotify, YouTube, Apple Music, Tidal и Broadcast. Цветовая индикация: зелёный — оптимально, жёлтый — понизят, красный — сильно приглушат.
Только Magic Master объединяет Reference Mastering + Dynamic EQ + Transient Designer + Streaming Preview в одном бесплатном веб-сервисе. LANDR и eMastered не имеют ни одной из этих функций одновременно.

FAQ

Часто задаваемые вопросы

Нужна ли регистрация для бесплатного мастеринга?
Нет. Бесплатный тариф работает без регистрации — просто загрузите файл и получите результат. До 3 мастерингов в день с одного устройства.
Какие форматы поддерживаются?
Загрузка: WAV, MP3, FLAC. Экспорт: WAV 16-bit (Free), а также MP3 320 kbps и FLAC 24-bit (Pro/Studio). Для MP3 требуется FFmpeg на сервере.
Что такое LUFS и зачем нормализация?
LUFS (Loudness Units relative to Full Scale) — стандарт громкости для стриминговых платформ. Spotify нормализует к −14 LUFS, Apple Music к −16, клубный звук к −9. Magic Master автоматически приводит трек к нужному уровню.
Хранятся ли мои файлы на серверах?
Нет. Файлы обрабатываются в памяти и удаляются сразу после отдачи результата. История мастерингов для Free-пользователей хранится только в браузере (localStorage).
Можно ли настроить цепочку обработки вручную?
Да, в тарифе Pro и Studio. Можно менять порядок модулей drag-and-drop, настраивать Amount для каждого, включать Linear Phase EQ, выбирать тип реверберации и M/S параметры.
Есть ли API для интеграции в мой инструментарий?
Да. Полная документация API доступна по адресу /docs. REST API позволяет программно запускать мастеринг, анализировать файлы и получать метрики. Для коммерческого использования — тариф Studio.
